在一个java文件中写上public class 文件名 或者 class 文件名

它才是一个会生成class并且能运行的java文件

比如HelloWorld.java文件中第一行写的就是 public class HelloWorld

写上public class 文件名 表示这是一个类 比如 public class HelloWorld 是HelloWorld类

一个文件还可以写成class 文件名        并且一个文件中还可以写多个class修饰的类

每一个class修饰的类都可以写main方法和输出某些文字
