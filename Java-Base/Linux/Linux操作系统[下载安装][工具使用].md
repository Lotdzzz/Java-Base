## Linux安装教程：
Linux属于跟Windows都是操作系统
Linux安全性要比windows好但是可视化界面缺一般般
首先准备好Linux的centos版本镜像（推荐7.x版本以上）：[Centos官网](https://www.centos.org/download/)
然后准备安装虚拟机软件Vmware(**选择VMware Workstation Pro 16.2.0 for Windows**)：[Vmware官网](https://customerconnect.vmware.com/cn/downloads/info/slug/desktop_end_user_computing/vmware_workstation_pro/16_0)

两款软件准备完就可以创建linux系统了：
